# Legal-Client-Management-Communication-Hub

A full-stack PERN (PostgreSQL, Express, React, Node.js) application designed to help law firms efficiently manage **cases**, **clients**, and **legal documents**, all coordinated through attorney workflows.

The system centralizes firm operations by allowing attorneys and staff to organize case files, manage client records, and securely store and retrieve documents from a single platform.
